HipGISAXS high-performance gisaxs

A massively-parallel high-performance x-ray scattering data analysis code

Latest News

  • May 27, 2016: HipGISAXS version 1.0 is out!

Supported Platforms

Clusters and supercomputers, or single processor systems (including your laptop). HipGISAXS has been optimized for high-efficiency with the following processor architectures:

  • Intel multi-core processors
  • AMD multi-core processors
  • NVidia GPUs (Tesla or Quadro)
  • Intel Xeon Phi processors (MIC). [NOTE: This support has been temporarily removed.]

Dependencies

  • UNIX-like based system
  • GNU GCC 4.7 or above, or Intel C++ Compilers 13 or above
  • Scons 2.0 or above
  • Boost 1.49.0 or above
  • TIFF Library (libtiff)
  • [Optional] (For multi-node parallel version only) OpenMPI 1.4.4 or above, or MPICH2, or MVAPICH2
  • [Optional] (For GPU version only) NVidia CUDA SDK version 5.5 or above
  • [Optional] Parallel HDF5 Library
  • [Optional] (HDF5 dependency) Zlib
  • [Optional] (HDF5 dependency) Szip Library

Download

You can download this project as a tarball or a zip file.

You can also clone the project with Git by running:

$ git clone https://github.com/HipGISAXS/HipGISAXS.git

This will enable you to easily update your copy to the latest version of HipGISAXS.

Installation

Use "scons" to compile the software and generate the application binary. For details please refer to the README file provided with the software.

Documentation

A user guide is included in the software package.
